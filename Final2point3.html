Hand Grinders

The first method of grinding your own coffee, has us using the standard hand grinder. Simply load the whole bean coffee into the top, make sure you're on a flat surface, and start grinding using the attached handle. It's important to note that this is a burr grinder, which is superior to a blade grinder. The burr grinder will make sure you have an even grind, for an even extraction when brewing coffee.

The Automatic Grinder

The other method you can use for grinding coffee, is to use an automatic grinder. You can either weigh out the amount of coffee beans you're using to brew and load them in the top of the hopper or you can dump the whole bag in there and use the hopper to store the beans until you run out. What makes this easier is that you can set the dial on the front for how many seconds you want it to grind. This is a trial and error process, but eventually you will get to the point where you can set it and leave it and ensure you will have the same amount every time. You will notice blade grinders are not mentioned here, the reasoning is simple, don't use a blade grinder.
